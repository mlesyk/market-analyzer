About

Why

The core of this website was created a few years ago as CLI tool which helps to find profitable orders in EVE Online game. I am a software developer and my recent need was to remember technologies I knew but not practiced for a long time and learn something new. Information is remembered better when you apply it in the real project instead of abstract theory or exercises. Thus, to kill two birds with one stone, I've decided to create this website, with all the technologies I planned to practice.

FYI

Integration using EVE Online Single Sign On (SSO) is optional and does not offer additional features for now. There are features in roadmap which will use SSO: private user search filters presets, search filters presets subscriptions for in-game notifications. These features are not implemented yet.
